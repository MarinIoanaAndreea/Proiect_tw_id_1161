Descriere proiect: 
Acest proiect reprezintă o aplicație web de tip Single Page Application (SPA) destinată gestionării eficiente și securizate a unei colecții personale de citate, fiecare utilizator având acces exclusiv la propriile date printr-un mecanism de autentificare bazat pe token (JWT). Sistemul oferă atât funcții de administrare și organizare a citatelor, cât și integrarea unui serviciu extern – Google Translate API – pentru traducerea automată a conținutului. Aplicația este dezvoltată pe arhitectura client–server, front-end-ul React interacționând cu un back-end RESTful implementat în Node.js. În continuare sunt prezentate funcționalitățile principale, tehnologiile utilizate și stadiul actual al proiectului.

Funcționalități principale:

înregistrare și autentificare utilizatori (JWT)

CRUD pentru citate (create/read/update/delete)

organizare pe categorii și marcarea ca favorite

integrare cu Google Translate API pentru traducerea citatelor

interfață front-end React care consumă un back-end RESTful

Tehnologii propuse

Backend: Node.js, Express, ORM (Prisma/TypeORM), PostgreSQL

Frontend: React, React Router, Axios

Autentificare: JWT

Serviciu extern: Google Cloud Translation API
